# Fraud Detector
fraud detection system using machine learning, designed to identify fraudulent transactions in real-time or batch mode. The project supports credit card fraud, e-commerce fraud, insurance fraud, or any tabular transaction data with a fraud label.

## Key Features

End-to-end pipeline: data ingestion → preprocessing → feature engineering → training → evaluation → deployment<br>
Handles severe class imbalance (fraud rates often <0.1%)<br>
Multiple models: XGBoost, LightGBM, CatBoost, Neural Networks, Isolation Forest, etc.<br>
Real-time inference API (FastAPI)<br>
Explainable AI with SHAP values<br>
Model monitoring and drift detection<br>
Dockerized deployment<br>
